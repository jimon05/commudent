# 인스타그램 시딩 운영안

## 1. 목적

인스타그램은 브랜드 인지도보다 "사람들이 어떤 손글씨 결과물에 반응하는지"를 빠르게 보는 실험 채널로 운영한다. 특히 저장, 공유, 댓글의 문장 제안, DM 문의를 통해 실제 사용 맥락을 수집한다.

## 2. 계정 포지셔닝

한 줄 소개:

> 타이핑한 문장을 내 손글씨처럼 바꿔주는 AI 필기체 서비스

프로필 구성:

- 이름: 서비스명 + AI 손글씨
- 소개문: "내 필체로 편지, 카드, 다이어리 문장을 만들어보세요. 베타 신청 중."
- 링크: 랜딩페이지 UTM 링크
- 하이라이트: Demo, Beta, FAQ, Letters, Creator

## 3. 콘텐츠 기둥

| 기둥 | 목적 | 형식 | 예시 |
|---|---|---|---|
| Before/After | 기능 즉시 이해 | 릴스, 캐러셀 | 타이핑 문장 → 내 손글씨 결과 |
| 변형도 비교 | 차별점 설명 | 캐러셀 | 같은 문장 0/50/100 변형 |
| 감성 문장 | 저장/공유 유도 | 이미지, 릴스 | "오늘 고마웠던 사람에게 보낼 한 줄" |
| 사용처 제안 | 구매/신청 맥락 만들기 | 캐러셀 | 생일카드, 다이어리, 청첩장, 감사장 |
| 제작 과정 | 신뢰 형성 | 스토리, 릴스 | 샘플 작성, 결과 QA, 삭제 정책 |
| 참여형 UGC | 댓글/DM 유도 | 스토리, 댓글 이벤트 | "내 글씨로 남기고 싶은 문장을 남겨주세요" |

## 4. 첫 3주 콘텐츠 캘린더

### Week 1: 이해시키기

| 요일 | 콘텐츠 | 목표 |
|---|---|---|
| 월 | 릴스: "타이핑한 문장이 내 손글씨가 되는 과정" | 첫 인지 |
| 화 | 캐러셀: 변형도 0/50/100 비교 | 기능 이해 |
| 수 | 이미지: 편지 문장 샘플 | 저장 유도 |
| 목 | 스토리: 베타 신청 Q&A | 신뢰 |
| 금 | 릴스: 생일카드 결과물 제작 | 사용처 제안 |
| 토 | 캐러셀: 데이터 권리/삭제 정책 | 불안 해소 |
| 일 | 댓글 이벤트: "내 글씨로 남기고 싶은 문장" | 참여 |

### Week 2: 사용처 넓히기

- 다이어리/문구 스티커 예시
- 가족 기록/부모님 글씨 보존 예시
- 캘리그라피 작가 협업 가상 사례
- 브랜드 감사장/초대장 예시
- 댓글 문장으로 만든 샘플 공개

### Week 3: 베타 전환

- 베타 신청자 모집 릴스
- "손글씨 샘플은 얼마나 필요할까?" FAQ
- 신청자 인터뷰에서 나온 사용처 익명 공유
- 베타 참여 혜택 안내
- 기관/크리에이터 협업 문의 CTA

## 5. 게시물 템플릿

### 릴스 템플릿: 전후 비교

- 0~2초: 타이핑된 문장.
- 2~5초: 손글씨 샘플 일부.
- 5~8초: 필체 결과물 등장.
- 8~12초: 변형도 조절 장면.
- 마지막: "베타 신청은 프로필 링크."

캡션 예시:

> 직접 쓰기엔 길고, 폰트로 쓰기엔 너무 차가운 말들이 있습니다.  
> 이제 타이핑한 문장을 내 손글씨처럼 남겨보세요.  
> 베타 신청은 프로필 링크에서 받고 있어요.

### 캐러셀 템플릿: 변형도 비교

1. 표지: "같은 문장, 다른 손맛"
2. 변형도 0: 정돈된 결과
3. 변형도 50: 자연스러운 결과
4. 변형도 100: 더 인간적인 흔들림
5. 사용처: 편지/카드/다이어리
6. CTA: "내 글씨도 만들어보고 싶다면 베타 신청"

### 참여형 템플릿

질문:

- "내 손글씨로 남기고 싶은 첫 문장은?"
- "부모님/연인/친구에게 손글씨로 보내고 싶은 말은?"
- "내 글씨가 굿즈가 된다면 어디에 쓰고 싶나요?"

운영:

- 댓글 문장 중 3~5개를 선정해 샘플 이미지로 제작.
- 선정 문장은 익명 처리.
- 스토리 투표로 어떤 결과물이 가장 자연스러운지 묻기.

## 6. 시딩 대상

우선순위는 팔로워 수보다 실제 반응과 맥락 적합성이다.

| 유형 | 예시 기준 | 제안 방식 |
|---|---|---|
| 다이어리/문구 계정 | 손글씨, 다꾸, 스티커 콘텐츠 | 무료 베타 + 공동 게시물 |
| 캘리그라피 작가 | 강의/작품/전시 활동 | 작가 필체 디지털 확장 샘플 |
| 편지/선물 계정 | 기념일, 커플, 가족 콘텐츠 | 감성 문장 샘플 공동 제작 |
| 한글/문화 계정 | 한글날, 글쓰기, 문화 캠페인 | 체험 캠페인 공동 운영 |
| 소규모 브랜드 | 감사장, 초대장, 패키지 카드 | 브랜드 손글씨 메시지 샘플 |

## 7. DM 제안문

### 크리에이터용

안녕하세요. 저희는 개인의 손글씨를 학습해, 입력한 문장을 그 사람의 필체처럼 변환하는 AI 필기체 서비스를 준비하고 있습니다.  
계정의 손글씨/문구 콘텐츠와 잘 맞을 것 같아, 작가님의 문장이나 필체로 짧은 데모 샘플을 만들어 공동 콘텐츠로 테스트해보고 싶습니다.  
상업 공개 전 베타 단계라 부담 없는 형태로 진행 가능하고, 결과물 사용 범위는 사전에 확인드리겠습니다.

### 기관/브랜드용

안녕하세요. 개인 필체를 AI로 재현해 편지, 카드, 감사장 등으로 활용하는 서비스를 준비 중입니다.  
한글/손글씨/기록/고객 커뮤니케이션 주제와 연결해 작은 체험형 파일럿을 제안드리고 싶습니다.  
20~50명 규모로 샘플 작성부터 결과물 제공까지 운영안을 드릴 수 있습니다.

## 8. 성과 지표

| 지표 | 의미 | 목표 |
|---|---|---|
| 저장률 | 감성 문장/사용처의 힘 | 팔로워 대비 5% 이상 게시물 발굴 |
| 공유율 | 바이럴 가능성 | 공유율 높은 문장 유형 파악 |
| 댓글 수 | UGC 참여 | 문장 제안 댓글 확보 |
| 프로필 클릭 | 랜딩 전환 관심 | 게시물별 UTM 비교 |
| DM 문의 | 파트너/베타 가능성 | 주 5건 이상 목표 |
| 베타 신청 | 실제 전환 | 콘텐츠별 신청 기여 추적 |

## 9. 운영 주의사항

- "과제 대필", "서명 생성", "진짜 손글씨처럼 속이기" 톤을 피한다.
- 부모님/고인 필체 등 민감한 콘텐츠는 동의와 권리 설명을 먼저 둔다.
- 샘플 공개 시 본인 동의를 받는다.
- AI 기술 과시보다 감정적 결과물을 중심으로 말한다.
- 댓글 문장을 결과물로 만들 때는 원문 작성자 동의를 확인한다.

## 10. 반복 개선 방식

매주 다음 질문으로 회고한다.

- 저장이 높은 문장은 어떤 감정인가?
- 신청으로 이어진 게시물은 어떤 사용처인가?
- DM에서 가장 많이 묻는 불안은 무엇인가?
- "내 글씨"와 "예쁜 손글씨" 중 어느 쪽에 더 반응하는가?
- 변형도 기능을 이해시키는 데 가장 좋은 시각 표현은 무엇인가?
